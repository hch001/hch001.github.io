深度匹配

函数

函数名 功能 用法 输出/注释
type(relation) 输出关系的类型 MATCH(a:Person)-[r]->() return distinct type(r) “IS_FRIEND_OF”
id(relation | node) 输出id编号，适用于节点和关系 MATCH (p) where id(p) = 1 return p.name -
exists() 判断模式是否在图中，属性是否在节点和关系中 MATCH (a:Person) where exists(a.age) return avg(a.age) -
all(…) 所有的元素都满足条件时返回true MATCH p=(a)-[]->(b) where all(x in nodes(p) where exists(x.name)) return p -
any(…) 有一个元素为真即可 MATCH (a:Person) where any(x in a.names where size(x) >10) a.names的类型是列表
None(…) 表示没有一个成立 - -
Single(…) 只有一个成立 - -
endNode(relation) 关系的终止节点 - -
startNode(relation) 关系的起始节点 - -
coalesce() 返回列表中第一个非null值 - -
head()/last() 返回列表中第一个/最后一个元素 - -
size(list | String) 返回长度或者大小 - -
length() 路径长度 - -
count() max() min() sum() 不解释 - -
filter(…) 列表过滤元素 (x in array where) -
extract(…) 抽取元素 extract(x in path | x.age) -
keys(node) 返回节点key的列表 - -
labels(node) 获取标签 - -
nodes(path) 获取路径中的节点 - -
relationships(path) 获取路径中的所有关系 - -
range(begin,end[,step]) 和python用法一样 - -
reverse(list) 反转列表 - -
reduce 迭代计算 (total=0 x in path/array | total+x.age) (初始化 迭代过程| 表达式)

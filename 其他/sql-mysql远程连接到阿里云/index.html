使用navicat连接阿里云mysql

Jul 5, 2020 · 1 min read

在阿里云上配置mysql遇到的错误，使用navicat

  • 修改阿里云的防火墙机制

添加自定义TCP 端口3306(默认)即可

  • mysql默认绑定了3306端口只能通过localhost访问

cd /etc/mysql/mysql.conf.d
vi mysqld.conf
注释掉里面的 bind-address=127.0.0.1

  • 改变mysql库user表中的某个用户(root)允许的主机
-- 举例
user mysql;
select host,user from user;
update user set host='%' where user.user='root';
  • Client does not support authentication protocol requested by server 参考
-- mysql版本更新后加密方式变化  
mysql -uroot -p  
ALTER USER 'root'@'%' IDENTIFIED BY 'password' PASSWORD EXPIRE NEVER; 
-- 由于上一步改过root的主机(从localhost改为%)
-- 所以这里要用 'root'@'%'，下同
ALTER USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY 'password';
FLUSH PRIVILEGES;

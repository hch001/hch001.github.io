实用工具--You-get简单使用

May 14, 2020 · 1 min read

简介 you-get

you-get 是github上的一个项目，是一个命令行程序，方便我们进行网络的媒体资源的下载

使用 pip 下载 you-get

pip install you-get # 应该提前安装有python，这里用的python3

安装 FFmpeg 依赖

这里用的是windows10，可以直接去官网下载 官网链接
我把资源上传了，免积分，嫌慢的可以去下载，链接:https://download.csdn.net/download/milanla/12419738

添加环境变量

下载完压缩包之后，直接解压。
将目录下的 bin 文件夹添加到环境变量 path 中即可

ffmpeg -version # 查看是否配置成功

简单的使用教程

以下载视频为例，windows平台下的，这里使用的方法是先把命令存在一个windows批处理的文件中，然后执行该文件，直接在 cmd 里输入命令也行，不过资源多的话还是批处理好一点。
详细用法参考:you-get中文说明或者you-get Github

def getUrls()->list: # 获取资源的呢url的方法，这里自行选择用爬虫还是直接到网页复制链接都可以
    pass # 这里略过

if __name__=='__main__':
    urls = getUrls()
    directory = "" # 要存放资源的文件夹
    with open(directory+"/urls.bat","w",encoding="ansi") as file:
        for url in urls:
            file.write("you-get -o {} --format=HD {} --debug ".format(directory,url))
    # 这里的 -o 指定输出目录，-O可以指定输出文件名
    # --format 指定视频的清晰度 --debug 选择调试模式 这些都是可选的
    # 使用 you-get -i url 可以知道视频的有哪些可选的清晰度

注意这里的 encoding 要是 ansi 的，而不是 utf-8 否则会出现中文乱码
